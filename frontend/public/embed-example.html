Embeddable Chat Widget

Add AI-powered chat to any website with just a few lines of code

Get Started in Seconds

Copy and paste this code into your website to add the chat widget

<!-- Add this to your website's <head> section -->
<script src="https://your-domain.com/widget.js" 
        data-agent-id="YOUR_AGENT_ID"
        data-position="bottom-right"
        data-color="#2563eb"
        data-title="Chat with us"
        data-subtitle="We're here to help!"
        data-greeting="Hello! How can I help you today?"
        data-branding="true">
</script>

Advanced Configuration

For more control, you can initialize the widget programmatically:

<script src="https://your-domain.com/widget.js"></script>
<script>
    // Initialize with custom options
    const widget = new SupportWidget({
        agentId: 'YOUR_AGENT_ID',
        position: 'bottom-right', // or 'top-right', 'bottom-left', 'top-left'
        primaryColor: '#2563eb',
        title: 'Chat with us',
        subtitle: "We're here to help!",
        greeting: 'Hello! How can I help you today?',
        showBranding: true
    });
    
    // You can also control the widget programmatically
    // widget.toggleChat(); // Open/close the chat
    // widget.sendMessage('Hello!'); // Send a message
</script>

Options

  • agentId (required): Your unique agent ID
  • position: Widget position (bottom-right, top-right, bottom-left, top-left)
  • primaryColor: Primary color for the widget (hex, rgb, or color name)
  • title: Title shown in the chat header
  • subtitle: Subtitle shown in the chat header
  • greeting: Initial greeting message
  • showBranding: Whether to show "Powered by" branding

Methods

  • toggleChat(show): Show or hide the chat
  • sendMessage(message): Send a message programmatically
  • addMessage({ content, isUser }): Add a message to the chat
